# Figma design export

This folder stores input and generated files for exporting selected Figma frames into local design assets.

## Files

- `nodes.example.json` - example file with Figma node ids.
- `nodes.json` - local file with real node ids. Create it from the example.
- `nodes/` - raw Figma node JSON exported by the script.
- `exports/` - optional place for manual Figma exports.
- `../screenshots/` - PNG screenshots exported by the script.
- `../specs/` - markdown summaries generated by the script.

## Prepare nodes.json

Copy `nodes.example.json` to `nodes.json` and replace ids with real frame node ids from Figma.

Recommended format:

```json
{
  "nodes": [
    {
      "id": "1:2",
      "name": "Splash"
    },
    {
      "id": "3:4",
      "name": "Login"
    }
  ]
}
```

The script also accepts a simple array:

```json
[
  "1:2",
  "3:4"
]
```

## Run export

Install `requests` if it is not available:

```powershell
python -m pip install requests
```

Set environment variables and run the script:

```powershell
$env:FIGMA_TOKEN="your_token"
$env:FIGMA_FILE_KEY="your_file_key"
python tools/export_figma_design.py
```

Do not commit real Figma tokens. The token is read only from `FIGMA_TOKEN`.
